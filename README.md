# 🚀 System Cleaning and Maintenance Tool

This is a smart, safe, and lightweight system optimization tool built with Python. It helps clean up temporary windows files to free up disk space and safely terminates heavy background applications to clear your RAM, giving your PC a breath of fresh air before gaming!

## 🛠️ Features
* **Safe Temp Cleaning:** Deletes useless temporary cache files without touching your personal data.
* **RAM Optimization:** Safely terminates blacklisted apps to instantly free up system memory.
* **Error Shield:** Built-in protection system (`try-except`) prevents crashes on locked system files or restricted processes.
* **Lightweight:** Runs instantly and leaves zero background footprint.

## 📦 Requirements (Gereksinimler)

This tool requires the `psutil` library to scan and manage background processes. You can install it easily using the included `requirements.txt` file:

```bash
pip install -r requirements.txt
