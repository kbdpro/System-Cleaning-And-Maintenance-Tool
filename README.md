# 🚀 System Cleaning and Maintenance Tool

This is a smart, safe, and lightweight system optimization tool built with Python. It helps clean up temporary windows files to free up RAM and disk space, giving your PC a breath of fresh air before gaming!

## 🛠️ Features
* **Safe Temp Cleaning:** Deletes useless temporary cache files without touching your personal data.
* **Error Shield:** Built-in protection system (`try-except`) prevents crashes on locked system files.
* **Lightweight:** Runs instantly and leaves zero background footprint.

## 🚀 How to Run (Nasıl Çalıştırılır?)

1. Make sure you have **Python** installed on your computer.
2. Clone or download this repository.
3. Open your terminal in the project folder and run:
   ```bash
   python main.py
